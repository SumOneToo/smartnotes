# SmartNotes

SmartNotes bootstrap for **PDCA-032 / milestone 032.1**.

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- ESLint

## Quick start

```bash
npm install
npm run dev
```

## Project structure

- `src/app` — routing and app shell
- `src/components` — reusable UI/layout blocks
- `src/features` — domain modules (`notes`, `editor`)
- `src/integrations` — external connectors/adapters
- `src/lib` — shared helpers/config
- `src/types` — base shared types
- `docs` — architecture notes
